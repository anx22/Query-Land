# UX Flows

> UI schlank; MCP gleichrangig (Master §3.3). UI = Übersicht, Umsetzung läuft agentisch (§2.4, §4.4).

## Hauptnavigation
Overview · Projects · Technical Audit · Keywords & Rank · Content & Opportunities · Backlinks · Reports · AI Visibility · Settings

## Schlüssel-Screens
- **Project Overview** — Visibility, organische Klicks/Impressionen, Health, Top-Opportunities, Risiken, letzte Crawls/Reports.
- **URL Dossier** — eine URL als vollständiges SEO-Objekt: Crawlstatus, Indexability, GSC-Leistung, Rankings, Queries, interne/externe Links, Content-Fit, Performance, Issues, Opportunities, **Source-Anker (§4.3)**, Historie.
- **Opportunity Board** — priorisierte Maßnahmen, Filter (Typ/Projekt/URL-Gruppe/Impact/Effort/Status), Evidence Drawer, Validation Drawer.
- **Technical Audit** — Crawl Runs, Issue Groups, URL Explorer, Crawl-Diff.
- **Content Workspace** — Opportunities, Briefings, Refresh-Kandidaten, interne Linkvorschläge, Snippet-Vorschläge.
